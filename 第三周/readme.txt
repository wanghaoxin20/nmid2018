@功能
------------------------------------------
#多线程模拟图书的借书与还书过程
------------------------------------------

@思路
------------------------------------------
#给借书线程加上对象锁borrowLock
#给还书线程加上对象锁bookStore和borrowLock
#当进行还书时,只能有一个线程持有bookStore和borrowLock
#还书完成,先把borrowLock释放,在释放bookStore
#当进行借书时,如果当前书籍量为0,释放borrowLock,循环获取borrowLock对象,直到书籍量不为0,操作完成后释放锁
