@功能
---------------------------------------
#TCP协议传输模拟
---------------------------------------


@思路
---------------------------------------
@服务端
#用ServerSocket对指定端口进行监听
#读取客户端发送的数据,判断要进行的操作
---------------------------------------
@客户端
#通过socket连接指定地址端口的服务端
#向服务器发送请求
---------------------------------------


@客户端的输入消息类型和请求的操作
---------------------------------------
@mes=>   发送文字消息
mes=>this is a test
--------------------------------------
@cmd=>   服务器执行cmd命令并返回结果
cmd=>echo HelloServer
--------------------------------------
@sendfile=>   向服务端发送文件
sendfile=>test.png
--------------------------------------
@getfile=>    从服务器获取文件
getfile=>test.png
--------------------------------------
@#shut 退出客户端,与服务端断开连接
#shut


*客户端
@Client/src/fouthweek/client/Client.java
---------------------------------------
#该类继承Thread
---------------------------------------
@变量
#host---服务端地址
#port---服务端端口号
#socket---客户端内的socket对象
#ISAddress---socket地址,包含ip和端口号
#din---socket的输入流
#dout---socket的输出流
#sendfilepath---传输文件的文件夹
#getfilepath---从服务端获取文件的存放位置
#input---获取命令行输入
-----------------------------------------
@方法
#Client---构造方法
#run---线程运行,通过输入请求信息,判断客户端要进行的操作,然后向服务器发送请求信息
#deMes---向服务端发送文字消息
#doCmd---向服务端发送cmd命令并让服务端执行,并接受返回结果
#doSendFile---向服务端发送文件
#doGetFile---从服务端获取文件,如果接收文件与当前文件夹内文件重名,则文件名添加一个+
#init_system---初始化系统,socket,ISAddress,input的初始化
#init_info---读取配置文件,获取变量的参数
------------------------------------------

*服务端
@Server/src/fourthweek/server
------------------------------------------
@Server.java
------------------------------------------
#该类继承Thread

@变量
#sf---对时间的格式化
#ServerTxt---Server.txt文件的路径
#recfilepath---接收文件的文件夹
#serverSocket---服务端的socket
#currentSocket---当前服务端接收的socket
#port---监听的端口
#excutor---线程池
#corePoolSize,maximumPoolSize,keepAliveTime---线程池的参数

@方法
#Server---构造方法,对服务端进行初始化
#run---线程运行,循环接受客户端,每接受到一个在线程池内启动一个Task线程
#init---服务器初始化,读取配置文件,初始化各参数
-------------------------------------------
@Task.java
-------------------------------------------
#处理每个客户端发过来请求的任务类
#该类实现Runable接口

@变量
#writelock---写入Server.txt的锁
#sf---对日期的格式化
#ServerTxt---Server.txt文件的路径
#din---当前任务socket的输入流
#dout---当前任务的socket的输出流
#socket---当前任务的socket对象
#recfilepath---传输文件的文件夹

@方法
#Task---构造方法,对Task进行初始化
#run---线程运行,对从客户端接受到的数据进行判断,再进行不同操作
#
#deMes---处理客户端的文字消息
#doCmd---处理客户端对服务端的cmd命令,并返回客户端结果
#doGetFile---处理客户端从服务端获取文件
#doRecFile---处理客户端发送文件给服务端,如果发送文件与当前文件夹内文件重名,则文件名添加一个+
#writeToTxt---将该次操作信息记录到Server.txt
#MesInfo---该次请求的信息
#init---初始化参数
---------------------------------------------
